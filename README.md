# MultiGeiger
Das Projekt "Multigeiger" wurde von Jürgen Böhringer entwickelt und ist auf seiner Webseite http://www.boehri.de beschrieben.

Die Dokumentation dazu ist im Verzeichnis 'doc' zu finden (incl. Bauanleitung) und der Sourcecode liegt im Verzeichnis 'src' als .ino-Datei und kann mit der Arduino-IDE übesetzt werden.
